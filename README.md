# digit-recognition-mnist
🔢 Handwritten Digit Recognition (MNIST)

📌 Description

This project recognizes handwritten digits (0–9) using a CNN model trained on MNIST dataset.

🚀 Features

- Digit classification
- High accuracy model
- Works on image input

🛠️ Technologies

- Python
- TensorFlow / Keras

📂 How to Run

1. Open in Google Colab
2. Run all cells
3. Test with image
